# Loona

Discord bot for Konserva

## Developing

1. Build the project

```bash
cargo build
```

1. Run in development mode

```bash
cargo run
```

## Building

1. Build an optimized release binary

```bash
cargo build --release
```

The compiled binary will be available at `target/release/loona`.
